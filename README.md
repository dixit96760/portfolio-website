# Portfolio Website

A modern and responsive portfolio website built using HTML, CSS, and JavaScript. It features smooth animations, interactive UI components, a custom cursor, and a clean design to showcase professional experience, skills, and projects.

## Live Demo

https://dixit96760.github.io/portfolio-website/

> Replace this URL with your GitHub Pages link after deployment.

---

## Features

- Responsive design
- Interactive animations
- Custom animated cursor
- Particle background
- Smooth scrolling navigation
- Professional portfolio layout
- Contact section

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
